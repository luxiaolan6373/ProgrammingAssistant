SetDict


函数简介:

设置字库,成功返回1,失败返回0

函数原型:

int  SetDict(index,file,pwd)

参数定义:

index 整形数:字库的序号,取值为0-9,目前最多支持10个字库
file 字符串:这里可以是字库的文件路径(相对全SetPath的路径或者绝对路径),也可以直接用字库里的内容.

pwd 字符串:字库密码，如果对字库进行过加密，要提供字库的字库密码。可空，默认为一个空的字符串。

返回值:

整形数:
0:失败
1:成功

示例:


ret = lw.SetDict(0,"test.txt")

注: 此函数速度很慢，全局初始化时调用一次即可，切换字库用UseDict
