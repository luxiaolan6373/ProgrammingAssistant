乐玩插件接口说明 v7.53

SetScreen


函数简介:

设置系统的分辨率 系统色深

函数原型:

int SetScreen(width,height,depth)

参数定义:

width 整形数: 新的屏幕宽度,0表示不改原来的值,默认为0

height 整形数: 新的屏幕高度,0表示不改原来的值,默认为0

depth 整形数: 新的系统色深,0表示不改原来的值,默认为0

返回值:

整形数:
0 : 失败

1 : 成功

示例:


ret = lw.SetScreen(1024,768,16)
