乐玩插件接口说明 v7.800

WriteString


函数简介:

对指定地址写入一段文本数据

 

int WriteString(addr,v,type)

 

参数定义:
addr 整形数: 要写入的地址

v     字符串: 要写入的文本 

type 整数型: 字符串类型,0= GBK字符串;1 =Unicode字符串 默认为0

 

返回值:

整形数:
0 : 失败

1 : 成功

 

示例:

ret = lw.WriteString(1234,"你好世界！")

 

注意,在读写之前一定要保证该对象使用过 SetMemoryProcess命令设置过读取的目标进程.参见(SetMemoryProcess.html)之后的所有内存相关命令都一样,不再提示!
